<p class="atribut-class"></p>

Digunakan untuk menetapkan satu atau lebih nama kelas CSS untuk elemen, memungkinkan penggunaan gaya yang sama untuk beberapa elemen. Atribut class dapat digunakan di semua elemen.



<p class="atribut-id"></p>

Menetapkan identitas unik pada elemen yang hanya bisa digunakan sekali per halaman. Biasanya digunakan dalam CSS atau JavaScript untuk memilih elemen secara spesifik. Atribut id dapat digunakan di semua elemen.



<p style="color: royalblue; font-family: 'Arial';></p>

Menambahkan gaya CSS langsung ke elemen. Atribut style dapat digunakan di semua elemen.



<p title="Atribut title"></p>

Menyediakan informasi tambahan tentang elemen. Biasanya ditampilkan sebagai tooltip saat pengguna mengarahkan kursor ke elemen. Atribut title dapat digunakan di semua elemen.



<p dir="rtl"></p>

Menentukan arah teks dalam elemen (left-to-right atau right-to-left). Atribut dir dapat digunakan di semua elemen.



<p contenteditable="true"></p>

Menentukan apakah konten elemen dapat diedit oleh pengguna. Atribut contenteditable dapat digunakan hampir di semua elemen.



<a href="index.html"></a>

Menetapkan URL atau tujuan tautan. Atribut href dapat digunakan di elemen <a>, <link>


<img src="image.jpg">

Menentukan sumber untuk konten media seperti gambar atau video. Atribut src dapat digunakan di elemen <img>, <script>, <iframe>, <audio>, <video>, <source>.


<img src="image.jpg" alt="Gambar.jpg">

| Menyediakan teks alternatif jika gambar tidak bisa ditampilkan. Atribut alt dapat digunakan di elemen <img>, <script>.


<input type="text">

Menentukan jenis konten atau elemen, seperti tipe input, tipe tombol, tipe file, dan lain-lain. Atribut type dapat digunakan di elemen <input>, <button>, <script>, <style>, <link>.



<input type="text" placeholder="Masukan Nama">

Menyediakan teks sementara yang ditampilkan di dalam input saat kosong. Atribut placeholder dapat digunakan di elemen <input>, <textarea>.



<input type="text" value="Ramdani Arvianto">

Menentukan nilai awal dari elemen input atau button. Atribut value dapat digunakan di elemen <input>, <button>, <option>.



<input type="text" value="Ramdani Arvianto" readonly>

Menentukan bahwa elemen input hanya dapat dibaca, tetapi tidak dapat diubah. Atribut readonly dapat digunakan di elemen <input>, <textarea>.



<input type="text" value="Ramdani Arvianto" disabled>

Menonaktifkan elemen sehingga tidak dapat digunakan oleh pengguna. Atribut disabled dapat digunakan di elemen <input>, <button>, <select>, <textarea>, <fieldset>, <option>.



<input type="text" minlength="1" maxlength="6">

Menentukan jumlah maksimum karakter yang dapat dimasukkan di dalam input. Atribut maxlength dapat digunakan di elemen <input>, <textarea>.
